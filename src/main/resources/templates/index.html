Welcome to Inventory Management Application


It is basic Inventory Management application created with:
- Java Spring Boot Web,
- Spring Data JPA & Hibernate,
- Thymeleaf,
- MySQL Database;
- Maven dependencies;
- HTML 5 & Bootstrap 4.
Has possibility to create, delete and update: Products, Suppliers, Customers, Employees and Orders.
